# Скриншоты для GUIDE.md

Положите сюда файлы с указанными именами — и картинки в `GUIDE.md` отобразятся
автоматически. Имена должны совпадать точь-в-точь (расширение `.png`).

|-----------|------------------------------|------------------|
| `08-vercel-deployments.png` | Список Deployments в Vercel: видно Production/Current и ветку `main` | Раздел 8.1 |
| `08-vercel-overview.png` | Обзор проекта Vercel: блок Production Deployment и адрес в Domains | Раздел 8.2 |
| `09-vercel-add-env.png` | Панель «Add Environment Variable»: поля Key/Value, тумблер Sensitive, Save | Раздел 9 |
| `09-vercel-env-list.png` | Список переменных проекта: `TELEGRAM_BOT_TOKEN`, `GROQ_API_KEY` | Раздел 9 |
| `10-botfather-bot-menu.png` | Меню бота в BotFather: Edit Info, Commands, **Mini Apps**, Bot Settings… | Раздел 10 |
| `10-botfather-mini-apps.png` | Экран **Mini Apps**: пункты Menu Button и Main App, Direct Links | Раздел 10 |
| `10-botfather-main-app.png` | Экран **Main App**: поле Enter URL, Launch Mode (Compact/Fullsize/Fullscreen), Save | Раздел 10.1 |
| `10-botfather-menu-button.png` | Экран **Menu Button**: предпросмотр кнопки, поле URL, текст кнопки, Save | Раздел 10.2 |
| `10-botfather-bot-settings-modes.png` | Страница Bot Settings с тумблерами (Inline/Group и т.п.) — пример, где Mini App НЕТ | Раздел 10, грабли №4 |
| `11-setwebhook-result.png` | Ответ браузера на `setWebhook`: `{"ok":true,...,"Webhook was set"}` | Раздел 11 |

> под нужными именами и поместите в эту папку.
