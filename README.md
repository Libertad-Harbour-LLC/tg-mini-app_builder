# Автошкола Тесла — Telegram Mini App

Мини-приложение для автошколы: категории и цены, расписание наборов,
запись на обучение и контакты. React + Vite + TypeScript, оформлено под
нативную тему Telegram (светлая/тёмная), mobile-first.

## Запуск

```bash
npm install
npm run dev      # дев-сервер на http://localhost:5173
npm run build    # прод-сборка в dist/
npm run preview  # предпросмотр сборки
```

## Что отредактировать

Весь контент — в `src/data.ts`:

- `SCHOOL` — название, телефон, адрес, ссылка на сайт, карта и **`botUsername`**
  (username бота без `@`, через которого открывается Mini App);
- `COURSES` — категории, цены, сроки (помечены `TODO`);
- `SCHEDULE` — ближайшие наборы;
- `PROMOS` — акции.

## Подключение к боту

1. Соберите приложение (`npm run build`) и разместите `dist/` на HTTPS-хостинге
   (или настройте Web App URL в @BotFather).
2. В @BotFather задайте Web App для бота / кнопку меню с URL приложения.
3. Форма записи (`src/components/EnrollForm.tsx`) отправляет заявку через
   `Telegram.WebApp.sendData(...)` — данные приходят боту, если Mini App открыт
   через **reply-keyboard кнопку** `web_app`. Для inline-кнопок `sendData`
   недоступен — предусмотрен фолбэк через `openTelegramLink`. При необходимости
   добавьте отправку заявки на свой backend в месте, помеченном в коде.

## Безопасность

Поле `Telegram.WebApp.initData` нужно валидировать на сервере (проверка
подписи по токену бота) перед доверием данным пользователя. Клиентский
`initDataUnsafe` использовать только для отображения.
